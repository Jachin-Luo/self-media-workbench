# 验收标准

## 模板

每项功能使用以下格式：

```text
前置条件：
操作步骤：
预期结果：
异常结果：
```

## 首版核心验收项

- 用户能够配置并测试 OpenAI 兼容接口。
- 用户能够手动刷新并查看热点。
- 用户能够保存素材并用于创作。
- 用户能够生成母稿及至少一个平台稿。
- 单个平台失败不影响其他平台结果。
- 用户能够编辑内容、上传图片并在分发页复制。
- 应用能够正确统计每日 Token。
- 应用能够完成整库备份和恢复。

详细验收条件随页面规格补充。

原型阶段回归结果见 [原型交互回归记录](./prototype-regression.md)。
