# 风险清单

| 风险 | 影响 | 当前措施 | 状态 |
|---|---|---|---|
| 公共热点 API 不稳定或变更 | 热点无法刷新 | 设计超时、重试、异常提示和可替换接口层 | 待设计 |
| OpenAI 兼容接口差异 | 生成或 Token 统计异常 | 统一适配层和结构校验 | 待设计 |
| AI 输出不符合字段结构 | 内容缺失或无法保存 | 结构化输出、修复与重试 | 待设计 |
| Windows/macOS 行为差异 | 安装、凭据或文件操作失败 | 双平台持续测试 | 待执行 |
| 本地数据库或图片损坏 | 用户数据丢失 | 整库备份、校验和异常恢复 | 待设计 |
| 热点正文抓取受限 | 无法获取参考正文 | 默认关闭、失败时退回榜单信息 | 已确定 |
| 长文和多素材导致上下文过大 | 成本增加或生成失败 | 长度预算、截断和摘要策略 | 待设计 |
