# 安全与隐私

## 已确认

- API Key 保存到操作系统凭据库，不进入普通配置或日志。
- 默认不长期保存完整 AI 提示词和原始响应。
- 调试模式默认关闭，完整请求响应可一键清除。

## 开发要求

- 日志必须过滤 API Key、Authorization 头和其他凭据。
- 恢复备份前必须校验文件格式和版本。
- 删除、清理和恢复等高影响操作必须二次确认。
- 外部链接正文读取需限制协议、大小、超时和重定向。
- 正文读取禁止访问本机、局域网、文件协议和非 HTTP(S) 地址，防止 SSRF。
- Renderer 禁止直接访问 Node.js、本地数据库和文件系统。
- Preload 只通过 contextBridge 暴露经过白名单定义的类型安全 IPC API。
