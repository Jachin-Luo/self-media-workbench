# 用户流程

## 主流程

```text
手动刷新热点
  → 选择聚合热点或输入自定义主题
  → 可选录入/勾选参考素材
  → 设置统一创作选项并勾选平台
  → 生成和编辑母稿
  → 生成各平台内容
  → 单独刷新或编辑内容模块
  → 上传各尺寸成品图
  → 在分发页复制内容
  → 手动标记分发状态
```

## 热点正文读取

```text
进入创作设置
  → 开启读取热点链接正文
  → 选择来源或选择排名最高 3 个
  → HTTP 读取与正文提取
  → 成功：保存项目快照
  → 失败：退回标题和摘要，可手动粘贴正文
```

## 平台失败与重试

```text
并行生成所选平台
  → 单个平台失败显示错误，不影响成功平台
  → 用户重试失败平台
  → 成功后建立或恢复对应分发任务
```

## 版本切换

```text
自动保存当前编辑稿
  → 用户点击保存版本
  → 原当前版本移动为上一版本
  → 编辑稿成为新的当前版本
  → 可只读查看上一版本
  → 恢复后进入未保存版本的编辑稿状态
```

## 删除、备份与恢复

```text
删除素材/项目 → 预览影响 → 二次确认 → 事务删除
创建备份 → 生成清单与校验值 → 完整性校验 → 保存压缩包
恢复备份 → 校验版本与文件 → 自动备份当前数据 → 覆盖并迁移 → 重启
恢复失败 → 回滚恢复前备份 → 显示失败阶段与日志位置
清理热点 → 选择日期范围 → 预览数量 → 二次确认 → 保留项目快照
```
