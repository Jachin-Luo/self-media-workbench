# 待确认事项

> 最近更新：2026-07-15

## 当前状态

M0 阶段暂无阻塞性待确认事项。

## 已收敛

1. 热点加载、空数据、部分平台失败使用独立状态提示；成功数据继续保留。
2. AI 联网搜索不纳入首版，后续版本再确定搜索服务、引用和缓存策略。
3. 搜索与筛选支持组合条件；无结果时展示清除条件建议，不自动改变用户筛选。
4. 备份恢复执行清单与校验值验证；高版本备份拒绝恢复，失败时回滚恢复前备份。
5. 自动保存失败固定显示重试入口，离开项目和退出应用前二次确认。
6. 最小窗口为 900×640；主快捷键使用 Windows Ctrl / macOS Command 映射。

## 后续版本议题

- AI 联网搜索服务、引用格式和缓存策略。
- 全局搜索及跨模块快捷跳转。
- 自动发布和图片生成模型。
