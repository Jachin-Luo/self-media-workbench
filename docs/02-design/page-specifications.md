# 页面规格

## 首次启动向导

- 第一步选择业务数据目录，验证通过后才能继续。
- 第二步配置 Base URL、API Key 和模型名，并提供测试连接。
- AI 配置步骤允许跳过。
- 跳过后进入热点页；需要 AI 的操作显示配置引导。

## 热点页

### 页面目标

查看当前及历史热点，并将热点转为素材或创作主题。

### 入口与出口

- 应用启动后默认进入该页面。
- 可进入热点详情、素材库新建页或新建创作流程。

### 页面区域

- 顶部页签：聚合热点、平台榜单。
- 统一时间筛选默认“今天”，支持昨天、最近 7 天、最近 30 天和自定义日期。
- 历史热点通过时间和搜索条件查询，不设置独立页签。
- 平台榜单内部按平台切换，不混排不同平台榜单。
- 右上角操作：刷新热点。
- 主区域：热点列表。

### 字段与操作

| 区域 | 字段/操作 | 规则 |
|---|---|---|
| 聚合热点 | 主题、综合热度、来源数量、趋势 | 同一事件的跨平台热点合并展示 |
| 平台榜单 | 平台、排名、标题、热度 | 按今日头条、百度、微博、抖音、知乎展示原始榜单 |
| 热点卡片 | 查看详情 | 打开热点详情 |
| 热点卡片 | 收藏为素材 | 将热点内容带入素材录入流程 |
| 热点卡片 | 开始创作 | 将热点作为创作主题 |
| 页面操作 | 刷新热点 | 由用户手动触发公共热点 API 请求 |
| 页面操作 | 重新聚合 | 原始榜单已保存但 AI 聚合失败时可单独触发 |

### 热点详情抽屉

- 从页面右侧打开，不离开当前榜单和筛选状态。
- 展示聚合主题、AI 摘要、综合热度和趋势。
- 展示各平台来源、排名、热度、原始标题和原始链接。
- 展示首次发现时间和最近更新时间。
- 提供收藏为素材和开始创作操作。

### 页面状态

- 加载中。
- 空数据。
- 正常数据。
- 部分失败。
- 完全失败。

### 确认与提示

- 刷新失败时保留现有数据并提示失败原因。
- 刷新过程先保存原始榜单，再执行 AI 聚合；两步分别显示状态。
- AI 聚合失败不影响平台榜单查看。
- 热点加载、空数据及部分平台失败的具体表现待确认。

## 后续页面模板

每个页面确认后按以下模板补充：页面目标、入口与出口、页面区域、字段与操作、页面状态、确认与提示。

## 创作模块

### 创作项目列表

- 展示项目名称、主题来源、已生成平台、更新时间和分发状态。
- 支持关键词搜索和状态筛选。
- 支持新建项目、复制项目和批量删除。
- 删除操作需要二次确认。
- 复制项目时复制当前编辑稿和参考快照，不复制历史版本及分发记录。

### 全局新建入口

- 左侧导航顶部固定“新建创作”。
- 直接点击创建空白草稿项目并进入创作设置。
- 从热点开始创作时自动带入热点快照；素材只在创作设置中选择。

#### 项目状态

| 状态 | 条件 |
|---|---|
| 草稿 | 尚未生成完整母稿 |
| 创作中 | 已有母稿，但所选平台内容未全部完成 |
| 待分发 | 所选平台必填字段全部完成 |
| 分发中 | 至少一个平台已复制或发布，但尚未全部发布 |
| 已完成 | 所选平台全部标记为已发布 |

生成失败显示独立错误标记，不单独作为项目状态。

### 项目详情

- 使用独立页面承载复杂创作流程。
- 包含创作设置、母稿、平台内容和图片素材四个页签。
- 顶部固定显示项目名称、自动保存状态、保存版本按钮和当前/上一版本入口。
- 创作设置包含主题、个人见解、参考素材、目标平台和统一创作选项。
- 创作话题允许完全自定义，不强制关联热点。
- AI 联网搜索显示为后续版本能力，首版禁用；用户通过热点、素材或手动输入补充资料。
- 支持搜索并关联多个热点。
- 参考素材通过可搜索弹窗选择，设置页只展示已选素材摘要。
- 所选热点和素材只在生成母稿时进入 AI 上下文。
- 热点主题进入创作设置后，提供默认关闭的“读取热点链接正文”选项。
- 开启正文读取后展示热点来源列表供用户多选，默认不选择，并提供“选择排名最高的 3 个来源”。
- 母稿页签负责生成与编辑母稿。
- 平台内容页签负责生成、编辑、刷新和重试各平台字段。
- 图片素材页签展示图片提示词，并保存用户按尺寸上传的成品图。

### 新建项目流程

```text
创作设置
  → 生成母稿
  → 用户确认或编辑母稿
  → 生成所选平台内容
  → 上传各尺寸成品图
```

- 四个页签始终可见，不强制锁定浏览顺序。
- 未生成母稿时可以查看平台内容页签，但生成平台内容的按钮不可用。
- 其他依赖前置数据的操作采用相同规则。

### 保存状态

- 所有输入和编辑内容自动保存。
- 停止输入约 1 秒后写入本地数据库。
- 顶部显示“保存中”“已保存”或“保存失败”。
- 保存失败且用户尝试离开页面时，弹窗提示并阻止无提示离开。
- 保存失败状态固定显示错误摘要、“重试保存”和“打开日志”操作；重试成功后恢复“已保存”。

### 版本规则

- 整个创作项目作为一个版本，包括创作设置、母稿、各平台内容、图片提示词和上传图片信息。
- 自动保存只更新当前编辑稿，不产生版本记录。
- 用户点击“保存版本”后，原当前版本整体成为上一版本，编辑稿成为新的当前版本。
- AI 重新生成、模块刷新、手动编辑和图片上传不会自行创建版本。
- 每个项目最多保存当前版本和上一版本。
- 上一版本默认只读，并提供“恢复为当前编辑稿”。
- 恢复后先进入编辑稿状态，用户点击“保存版本”后才完成版本变更。

### 平台内容布局

- 内部按用户已选平台设置页签。
- 每个平台页签纵向排列该平台的字段卡片。
- 字段卡片包含字段名称、编辑区、复制、AI 刷新、生成状态和错误重试。
- 生成过程不展示逐字流式内容，只展示当前阶段和耗时；校验通过后一次性显示结果。
- 生成结果超限时后台自动压缩和裁剪，用户只看到符合平台限制的最终内容。
- 页面顶部提供“生成全部所选平台”。
- 不提供“一键刷新全部”，防止误覆盖用户已经编辑的内容。
- 微信公众号和今日头条正文使用 TipTap 富文本编辑器。
- 小红书和抖音使用纯文本编辑器，不展示复杂排版工具。

### 图片素材布局

- 图片只存在于所属创作项目内，并按目标平台和用途分组。
- 每种封面规格使用一张卡片，只保留一张当前图片。
- 封面卡片包含用途、比例、建议尺寸、提示词、复制、刷新、上传、预览、替换和删除。
- 内容配图允许多张，支持上传、预览、排序、替换、删除和可选备注。
- 每张内容配图保留对应提示词及排列顺序。
- 系统按正文篇幅和段落结构自动生成最多 8 个内容配图提示词。
- 用户可新增、删除或单独刷新提示词。
- 内容配图可选绑定到正文某一段之后；未绑定时只按图片顺序管理。
- 分发页按绑定位置显示图片占位提示，帮助用户手动发布时定位。

## 内容分发页

### 列表结构

- 以创作项目作为父行。
- 项目父行展示项目名称、更新时间和已发布平台数/平台总数。
- 展开父行后，以子行展示各平台分发任务。
- 平台子行展示平台、状态、更新时间、发布时间和操作。
- 支持按状态、平台和关键词筛选项目及其平台任务。
- 点击平台子行后打开右侧分发详情抽屉。

### 分发详情抽屉

- 按平台发布字段顺序展示内容。
- 每个字段提供独立复制按钮。
- 不提供“一键复制全部文本”。
- 图片提供单独复制和打开所在位置。
- 任一字段复制成功后，该平台任务自动变为已复制。

## 统计页

### 数据总览

- 热点总数。
- 素材总数。
- 创作项目数。
- 已发布平台任务数。

### Token 趋势

- 按日期展示输入 Token、输出 Token 和总 Token。
- 时间范围支持最近 7 天、30 天、90 天和自定义日期。
- 图表下方提供每日明细表。
- 首版不提供平台占比和模型费用分析。

## 日志页

- 分为操作日志、错误日志和 AI 调用三个页签。
- 统一展示时间、类型、模块、结果和摘要。
- 支持日期、模块、结果筛选和关键词搜索。
- 错误日志可展开查看错误堆栈。
- AI 调用可查看模型、耗时、输入 Token、输出 Token 和总 Token。
- 日志保留 30 天后自动清理，用户也可手动清空。

## 设置页

### AI 模型

- Base URL、API Key、模型名和测试连接。

### 数据管理

- 数据目录、整库备份、恢复和热点清理。
- 热点清理提供 30 天前、90 天前、自定义日期前和全部，默认选择 30 天前。
- 执行前展示预计删除条数并二次确认。

### 日志与调试

- 日志保留天数、调试模式和清空日志。

### 关于

- 应用版本、更新检查和开源许可。

正式版不展示提示词模板；提示词模板只在开发模式中可见。

### 状态规则

- 状态包括待分发、已复制和已发布。
- 点击任一内容字段的复制按钮后，平台分发任务自动标记为已复制。
- 已发布由用户手动标记，并可选填写发布时间和作品链接。
- 已发布后创作内容发生变更时，状态保持不变，并显示“发布后有修改”。

## 素材库页

### 页面目标

保存和管理创作参考文本，并将素材带入创作流程。

### 页面结构

- 主体使用素材列表。
- 新建和编辑使用右侧抽屉，不跳转独立页面。
- 列表展示标题、内容摘要、标签、创建时间和最近使用时间。
- 支持关键词搜索；标签为自由文本参考信息，不建立标签库。

### 操作

- 新建素材：标题和正文必填，标签可选。
- 素材摘要：直接截取正文前部，不调用 AI 生成。
- 编辑素材：在右侧抽屉修改。
- 删除素材：二次确认后直接删除。
- 批量删除：选择多条素材后执行，二次确认后直接删除。
