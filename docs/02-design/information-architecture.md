# 信息架构

## 已确认一级导航

```text
新建创作（固定主操作）
热点（默认页）
素材库
创作
内容分发
统计
日志
设置
```

不设置独立工作台首页。

## 已确认内部结构

- 素材库使用列表与右侧编辑抽屉。
- 创作使用项目列表与独立项目详情页，详情包含四个固定页签。
- 内容分发使用项目父行、平台子行和右侧详情抽屉。
- 统计、日志和设置使用独立一级页面，不再增加工作台概览。
- 桌面窗口最小尺寸为 900×640；达到最小宽度时仍保留完整导航文字和新建创作入口。
- Windows 与 macOS 使用相同信息架构，仅主快捷键修饰符分别使用 Ctrl 和 Command。

## 全局新建创作

- 左侧导航顶部固定显示“新建创作”按钮。
- 点击后创建草稿项目并进入创作设置。
- 热点创作入口复用全局新建流程，并自动写入热点快照；素材只在创作设置中选择。

## 热点

```text
热点
├─ 聚合热点
└─ 平台榜单
   ├─ 今日头条
   ├─ 百度
   ├─ 微博
   ├─ 抖音
   └─ 知乎
```

两个页面共享时间和关键词筛选；历史热点通过筛选直接查询，不设置独立页签。

## 素材库

```text
素材库列表
└─ 右侧新建/编辑抽屉
```

## 创作

```text
创作
├─ 创作项目列表
└─ 项目详情
   ├─ 创作设置
   ├─ 母稿
   ├─ 平台内容
   └─ 图片素材
```

版本切换和保存状态固定显示在项目详情顶部。

## 内容分发

```text
项目父列表
└─ 平台分发任务子列表
   └─ 右侧分发详情抽屉
```

## 设置

```text
设置
├─ AI 模型
├─ 数据管理
├─ 日志与调试
└─ 关于
```
