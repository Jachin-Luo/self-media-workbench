# 设计规范

## 首版基线

- 主色使用紫色系，成功、警告和错误分别使用绿、橙、红语义色。
- 正文字体优先使用 Inter、苹方和微软雅黑；基础字号 14px。
- 基础圆角 8px，卡片圆角 11px，弹窗圆角 13px。
- 页面间距以 4px 倍数递进，主要内容区使用 24～28px 边距。
- 一级导航、按钮、表单、表格、卡片、抽屉、弹窗、状态提示和 Toast 使用统一语义样式。
- 所有数据页面必须覆盖加载、空数据、无搜索结果、部分失败、完全失败和禁用状态。
- 弹窗打开后聚焦首个控件，Tab 焦点限制在弹窗内，Esc 关闭可取消弹窗并恢复原焦点。
- 桌面窗口最小尺寸为 900×640；Windows 使用 Ctrl、macOS 使用 Command。

## 已确认

- 首版支持浅色、深色和跟随系统三种主题。
- 所有组件从设计阶段覆盖两种颜色模式。
