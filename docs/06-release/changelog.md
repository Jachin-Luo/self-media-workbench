# 版本与变更记录

## Unreleased

### Added

- 建立项目文档框架。
- 新增覆盖七个核心模块的可交互高保真原型。
- 新增首次启动数据目录与 AI 配置向导。
- 新增热点正文来源选择、营销目的和素材参考级别交互。
- 新增分发发布时间、作品链接和发布后修改状态。
- 新增热点部分失败、平台生成失败重试和正文读取失败演示状态。
- 新增项目级图片提示词、上传图片、排序和版本快照状态。
- 新增可重复执行的原型自动化交互回归脚本和回归记录。
- 建立 Electron 正式工程及类型安全的 Main、Preload、Renderer、Shared 分层。
- 实现首次启动工作区选择、目录初始化、路径持久化和启动验证链路。
- 新增工作区异常恢复页，支持重新选择、重试和退出。
- 实现首次启动 AI 配置、最小模型连接测试和跳过流程。
- 使用 Electron safeStorage 加密保存 API Key，普通 AI 配置保存在工作区。
- 接入 better-sqlite3、Drizzle ORM、编号迁移、WAL 和外键配置。
- 新增应用配置、操作日志、错误日志、AI 调用和每日 Token 统计基础表。
- AI 连接测试用量开始写入 AI 调用记录和按 Asia/Hong_Kong 汇总的每日 Token 统计。

### Changed

- 创作项目改为使用独立的平台内容状态。
- 未生成母稿时禁止生成平台内容，未生成平台内容时不建立分发任务。
- 修正各平台图片规格，并优化 900px 窄窗口导航。
- 搜索、状态筛选、平台筛选、日志筛选和统计时间范围改为真实交互。
- 设置项改为自动保存，并与首次启动配置保持一致。
- 项目状态和统计指标改为根据当前项目及分发任务实时计算。
- 收敛首版联网搜索、备份兼容、保存失败、窗口尺寸和快捷键决策，完成 M0。

### Fixed

- 修复空白项目显示其他项目平台稿的问题。
- 修复复制字段未更新分发状态的问题。
- 修复抖音封面比例错误和微信公众号分享图缺失问题。
- 修复项目改名后分发名称不同步、素材全选无法取消和搜索输入失焦问题。
- 修复连续保存版本时当前版本与上一版本语义错误的问题。
