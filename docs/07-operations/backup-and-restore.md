# 备份与恢复

## 备份范围

- 热点数据。
- 素材。
- 创作项目和版本。
- 上传图片。
- 分发记录。
- 应用配置，但不包含可直接读取的明文 API Key。
- `workspace.json` 工作区标识和格式版本。

## 排除内容

- 日志。
- 临时文件。
- 历史备份文件。
- API Key。

## 已确认流程

- 导出为单个压缩备份文件。
- 备份先生成到工作区 `backups` 目录，完成后允许用户另存到其他位置。
- 恢复前自动创建当前数据备份。
- 恢复采用整体覆盖，不合并新旧数据。
- 恢复完成后重启应用。
- API Key 不包含在备份文件中。
- 备份需要包含当前编辑稿、当前版本和上一版本所引用的全部图片资产。

## 待设计

- 备份包内部格式和版本号。
- 完整性校验和损坏处理。
- 跨 Windows 与 macOS 恢复的路径兼容。
